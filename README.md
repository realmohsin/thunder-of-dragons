1. Install and use the correct version of Node and NPM (https://nodejs.org)

2) Install dependencies

   ```sh
   npm i
   ```

3) Configure the database

   ```sh
   cd backend
   npm run config-db-dev
   ```
4) Start the backend server

   ```sh
   npm run dev
   ```

5) Start the frontend server

   ```sh
   cd ../frontend
   npm run dev
   ```
